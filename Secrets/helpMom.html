Useful Mac Apps, Utilities, and Gestures

Fresh Install:

Install Brew:

  • brew.sh
    • Copy and paste the install link
    • You probably won’t ever use this. Dad would be proud and would promptly begin mansplaining.
    • It’s a package manager, an app store without a GUI
      • Package and app mean the same thing here
    • Used to get dozer the menu bar organization thing
    • Things to note:
      • If “brew install …” isn’t working and the package is listed on the website, try “brew install --cask …”

Instal Chrome:

  • google.com/chrome
  • Or: “brew install google-chrome” in terminal (no quotes included, goes for anything else terminal related.)

Install Magnet:

  • Magnet in the App Store
  • Window manager
  • Things to note
    • Start at login

Install Google Drive system extension:

  • google.com/drive
  • things to note
    • Choose mirror files during setup, this keeps a local copy of your google drive on your computer, so working with those files is much faster, and SOOOOO much less annoying.
    • Start at login

Install Zoom:

  • zoom.us
  • Or: "brew install zoom"

Install Teams

  • microsoft.com/teams
  • Or: "brew install microsoft-teams"

Tips

Keyboard

Frequently used

Navigation

  • Cmd + tab: cycle through open apps
  • Cmd + shift + tab: cycle back
  • Cmd + `: cycle between open windows of focused app
  • Cmd + `: cycle backwards
  • Option + right or left arrow: Move spaces

System

  • Cmd + Space: Spotlight search
  • Cmd + ,: Opens preferences pane of most apps

Symols and Emoji

  • Tap the globe/fn button to get an emoji viewer
  • Holding option gives tons of symbols option + shift gives even more

Safari, Chrome, and Other Browsers

  • Cmd+T: new tab
  • Cmd + W: Close tab
  • Cmd + Shift + T: Restore last closed tab

Trackpad

Gestures

Three Fingers

  • Up: Mission Control (show all open windows in current workspace
  • Down: Windows open under current app
  • Double tap and slide up on second tap: Accessibility zoom in
  • Double tap: Accessibility zoom enable or disable

Two Fingers

  • Slide in from right side: Notification center
    • Slide right with 2 fingers again to dismiss
  • Slide left or right to go back and forth in browser windows (I have this disabled; I find that it makes navigating web pages that are zoomed fucking infuriating

Window and Workspace Management

Mission Control

  • Adding more Spaces: Done by going into mission control, moving your mouse up to the area where you see Desktop, it once you have moused over it, it will enlarge, there is then a plus icon directly to the left
  • Deleting spaces: Mouse over the top left corner and an X should appear, this simply moves any open windows to your main desktop, and will not close those windows

Magnet

Keyboard Shortcuts

  • Ctrl + Option+:
    • Up: moves a window to the top half
    • Down: move a window to the bottom half
    • Left: moves window to the left
    • Right: moves window to the right
    • J: bottom left quarter
    • K: Bottom right quarter
    • U: Top left quarter
    • I: Top right quarter
    • G: Right-most vertical third
    • F: Middle vertical third
    • D: Left-most Vertical third
    • Enter: Maximize
  • Other ways to use magnet
    • Just drag the window to a specific corner, play around with it a little.

Glossary

  • Spaces: additional virtual desktops found in mission control
  • Mission control: shows all open apps on current desktop
  • Launchpad: iOS style app library
  • System preferences: settings, soon to be system settings
  • Spotlight: System wide search
  • I/O (Input and Output): PORTS and wireless interfaces!!!
